# Сортировка слиянием

Выполнила:

 - Бастракова Ксения
 - ННГУ, ИИТММ, каф. МОСТ, группа 381603м4

## Задание

Реализовать класс для сортировки с поддержкой следующих возможностей:
 - Сортировка целых и вещественных чисел.
 - Сортировка по возрастанию и убыванию.
 - Устойчивая сортировка значений произвольных типов с заданным компаратором.

Все необходимые математические сведения можно найти в статье
[Merge sort][mergesort] на сайте Wikipedia.

## Документация разработчика

Диапазон значений для сортировки задается двумя итераторами `begin` и `end`. Осуществляется сортировка значений в диапазоне `[begin, end)`, результаты сортировки записываются на место вхоных значений. Итераторы должны удобвлетворять концепции [RandomAccessIterator][randomaccessiterator].

Класс предоставляет два метода, являющиеся шаблонами функций:
- Шаблон, принимающий пару итераторов. Параметром шаблона является тип итераторов. Для сравнения используется стандартный компаратор.
- Шаблон, принимающий пару итераторовы и компаратор. Параметрами шаблона являются тип итераторов и тип компаратора.
